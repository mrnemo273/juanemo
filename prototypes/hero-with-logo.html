Experiment 03
Generative Typography

About This Experiment

The JUANEMO wordmark uses Roboto Flex's 13 variable font axes to create a generative identity. Each letter independently drifts to random weight and width values on a timed cycle — hold, shift, hold. The wordmark is never the same twice.

The small logo in the top left uses the same system but frozen — a single random configuration set on page load. Every visit produces a unique mark. Refresh to see a new one.
